Protecting forms from spam without captcha or api Using only javascript, php is detected by spam robots

== Description ==

Protecting a form from being submitted with too many POST parameters, such as spam bots, forms without captcha or comments without default protection in wordpress. Detection of the robot sending a POST request and deleting it Preventing the sending of the message to email the spam created by the robot. There is interaction with the plugin contact form 7 etc. Messages from robots are not sent to the mail. Messages are now being deleted. Only a living person can send a letter from the form.

== Installation ==

1. Upload the entire `unifree-protect-forms-wp` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).
